# Ultimate Stealth Nuke Bot

**FOR EDUCATIONAL PURPOSE ONLY**

## Fitur
- Human-like behavior (typing indicator, random breaks)
- Dynamic delay + jitter
- Multi-token rotation with health check
- Proxy support
- Randomized messages + reply chance
- Anti-rate limit

## Cara Install

1. Clone repo
2. Install dependencies: `pip install -r requirements.txt`
3. Edit `tokens.txt` (isi token bot)
4. Edit `proxies.txt` (opsional)
5. Jalankan: `python ultimate_stealth_nuke.py`
